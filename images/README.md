# Images Folder

Esta carpeta contiene todas las imágenes y videos utilizados en el sitio web CAMEX-1.

## Estructura sugerida:

```
images/
├── hero/           # Imagen/video principal para el hero
├── galeria/        # Fotos de la expedición
├── equipo/         # Fotos del equipo científico
├── logos/          # Logos de instituciones y sponsors
└── iconos/         # Iconos y elementos gráficos
```

## Formatos recomendados:
- **Imágenes**: JPG (fotografías), PNG (logos con transparencia), WebP (optimizado)
- **Videos**: MP4 (H.264), WebM

## Notas:
- Optimizar todas las imágenes antes de subirlas
- Usar nombres descriptivos (ej: `vernadsky-estacion-exterior.jpg`)
- Mantener tamaños razonables para web (máx. 2MB por imagen)
